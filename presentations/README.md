# Carbon_GDP_Presentation 生成说明

1. 环境准备
   - Python 3.8+
   - 安装依赖: pip install python-pptx

2. 准备图片（可选）
   - 若要包含真实图片，请将三张图片保存并命名为 image1.png, image2.png, image3.png 放在脚本同目录

3. 运行脚本
   - 在命令行中运行:
     python generate_presentation.py
   - 成功后会生成 Carbon_GDP_Presentation.pptx

4. 占位 PPTX 说明
   - 本仓库同时包含一个占位文件 presentations/Carbon_GDP_Presentation.pptx。
   - 该占位文件仅用于占位与快速下载，如果你希望得到包含真实图片的最终 PPTX，请把图片上传到仓库同目录或把图片发给我，我会重新生成并推送最终版本。
